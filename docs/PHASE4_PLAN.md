# Phase 4 规划：智能化与易用性提升

## 核心目标
从「能用」到「好用」，降低嵌入式电路仿真学习门槛。

## 任务清单（按优先级）

### 1. 向导式电路搭建（Wizard System）
- 新手引导：分步搭建第一个电路
- 常见电路模板一键加载（LED驱动、电源滤波、放大器等）
- 元件推荐：根据已选元件智能推荐下一个
- 连线引导：高亮可连接的端口

### 2. 智能错误诊断（Circuit Diagnosis）
- 检测常见电路问题：
  - 悬空输入引脚
  - 短路检测
  - 电源/地未连接
  - 元件参数超出范围
  - 信号完整性警告
- 提供修复建议（一键修复按钮）
- 实时错误提示（红线标记问题元件）

### 3. 模板电路库（Template Library）
- 基础电路：分压器、RC滤波、LC振荡
- 放大器：共射/共基/共集、差分、仪表放大器
- 电源：线性稳压、开关电源、LDO
- 数字：触发器、计数器、状态机
- 传感器接口：温度、光敏、压力
- 通信：SPI/I2C/UART接口电路

### 4. 传感器元件库
- 温度传感器（NTC热敏电阻、DS18B20模型）
- 光敏传感器（光敏电阻、光电二极管）
- 压力/力传感器
- 加速度计模型
- 每个传感器包含：
  - 电气等效模型
  - 参数可配置（灵敏度、范围、响应时间）
  - 典型应用电路推荐

### 5. 电源管理IC
- 线性稳压器（7805, LM317）
- LDO稳压器
- DC-DC转换器模型
- 电池模型（充放电曲线）
- 电源监控IC

### 6. 中文文档系统
- 搜索功能
- 分类浏览（入门、元件、仿真、高级）
- 交互式示例
- API参考文档

### 7. 用户体验优化
- 快捷键体系完善（与主流EDA软件对齐）
- 右键菜单系统
- 拖拽元件时显示辅助线
- 元件旋转时保持连线智能调整
- 仿真进度条和预估时间
